    A Propos

    Contenu

    Vous trouverez sur ce site le contenu du cours de NSI en classe terminale du lycée Mémona Hintermann-Afféjee. Chaque chapitre se compose :

    • d'activités
    • d'un diaporama de cours au format pdf
    • d'exercices

    Remerciements

    • A la liste de discussion NSI qui a été une source d'inspiration constante dans l'écriture de ce cours (mention spécial au créateur de la liste et aux contributeurs réguliers).
    • Au forum des enseignants NSI.
    • Aux collègues qui avant moi ont mis en ligne leurs propres cours et notamment :
      • Le site de David Roche
      • Le github de Gilles Lassus
      • Le site du Lycée la Martinière
    • A Sebastien Hoarau pour les conseils et la relecture du contenu.

    Contact

    Mon adresse mail : fnativel2[at]ac-reunion.fr

    Utilisation

    Ce cours est construit pour être utilisé en classe, les élèves disposent d'un ordinateur fonctionnant sous Ubuntu et équipé des logiciels nécessaires. Cependant, une utilisation autonome à la maison est possible à condition d'installer les applications suivantes.

    Applications

    Applications gnu/Linux Windows MacOS
    Python
    VS Code
    Jupyter
    Filius
    Gameshell

    Cliquer sur une application pour accéder à sa page et l'installer.
