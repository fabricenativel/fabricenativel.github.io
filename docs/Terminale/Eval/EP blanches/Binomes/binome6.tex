\vspace{0.25cm}
\textbf{Enoncé}

Ecrire une fonction {\tt consecutifs} qui prend en argument une liste non vide et renvoie {\tt True} lorsque deux éléments consécutifs de cette liste sont identiques et {\tt False} sinon.


\textbf{Exemples :}
\begin{lstlisting}
>>> consecutifs([12, 8, 11, 8, 20, 15])
False
>>> consecutifs([7, 13, 13, 8, 10])
True
>>> consecutifs([19, 11, 2, -10, 19, 11, 2])
False
\end{lstlisting}
